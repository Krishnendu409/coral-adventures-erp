# Coral Adventures ERP

A fully integrated ERP and operational dashboard for Coral Adventures. 

## Project Structure

For ease of use, all user-facing directories are located directly in the root folder:

- **`/incoming`** — Drop filled Excel files (Reservations, Finance, Captain, etc.) here for the system to import.
- **`/generated`** — Blank Excel templates generated by the system will appear here.
- **`/reports`** — End-of-day or financial PDF/CSV reports will be exported here.
- **`/exports`** — General system data exports will appear here.

*(Note: Internal databases, logs, and configuration are safely hidden away in the `/data` directory).*

---

## Getting Started

### 1. Installation
Run the included installation script to install dependencies and initialize the database automatically:
```bash
install.bat
```
*(Under the hood, this runs `npm install` and `npm run db:migrate`)*

### 2. Seeding Dummy Data (Optional)
If you want to view the dashboards with pre-populated dummy data for testing purposes, run:
```bash
npm run db:seed:synthetic
```

### 3. Start the System
Run the included start script to boot up the web application and the automated import engine:
```bash
start.bat
```
*(Under the hood, this runs `npm run dev`)*

The dashboard will be available at **http://localhost:3000**.
